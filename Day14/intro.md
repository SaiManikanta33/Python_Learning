    What are Magic (Dunder) Methods?
Magic methods are special methods surrounded by double underscore(__). python automatically calls them in specific situations.
 
 some common dunder methods:

    Method          Purpose
   __init__        Constructor
   __str__         User-friendly string representation
   __repr__        Developer - representation
   __len__         Defines behavior of len()
   __add__         Defines behaviour of '+'
   __eq__          Defines behaviour of '=='
